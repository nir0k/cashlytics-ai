<task type="auto">
  <name>Add AUTH_SECRET to docker-compose.yml</name>
  <files>docker-compose.yml</files>
  <action>
Read the current docker-compose.yml and add AUTH_SECRET to the app service's environment section.

The AUTH_SECRET is required by Auth.js v5 for JWT encryption. It should be:

1. Added to the `environment:` section of the `app` service
2. Use variable substitution: `AUTH_SECRET: ${AUTH_SECRET}`

Example addition:

```yaml
services:
  app:
    environment:
      - DATABASE_URL=postgres://...
      - NEXT_PUBLIC_APP_URL=...
      - AUTH_SECRET: ${AUTH_SECRET} # Add this line
```

Do NOT add a hardcoded secret value — use environment variable substitution so users must provide their own secret.
</action>

</task>
