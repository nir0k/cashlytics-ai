<task type="auto">
  <name>Task 1: Add userId FK to all 8 data tables (nullable)</name>
  <files>src/lib/db/schema.ts</files>
  <action>
Add a nullable `userId` foreign key column to each of the 8 data tables. Place the column after `id` for consistency.

For each table, add:

```typescript
userId: uuid("user_id").references(() => users.id, { onDelete: "cascade" }),
```

Tables to update:

1. **accounts** - After `id`, before `name`
2. **categories** - After `id`, before `name`
3. **expenses** - After `id`, before `accountId`
4. **incomes** - After `id`, before `accountId`
5. **dailyExpenses** - After `id`, before `accountId`
6. **transfers** - After `id`, before `sourceAccountId`
7. **documents** - After `id`, before `expenseId`
8. **conversations** - After `id`, before `title`

CRITICAL:

- Keep the column NULLABLE (no `.notNull()`) - existing data has no userId yet
- Use `onDelete: "cascade"` so user deletion removes their data
- Column name in SQL is `user_id` (snake_case), TypeScript property is `userId` (camelCase)
  </action>

  </verify>
  <done> - All 8 data tables have nullable userId FK column - All FKs reference users.id with cascade delete - No tables have NOT NULL constraint yet
  </done>
