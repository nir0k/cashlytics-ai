```yaml
  - NEXT_PUBLIC_APP_URL=${NEXT_PUBLIC_APP_URL}
  - AUTH_SECRET=${AUTH_SECRET}
  - SMTP_HOST=${SMTP_HOST}
  - SMTP_PORT=${SMTP_PORT}
  - SMTP_USER=${SMTP_USER}
  - SMTP_PASS=${SMTP_PASS}
  - SMTP_FROM=${SMTP_FROM}
  - APP_URL=${APP_URL}
```

DO NOT add default values (e.g., `${SMTP_HOST:-}`) - let the app handle missing vars gracefully.
