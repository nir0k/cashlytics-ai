**ResetPasswordState type:**

```typescript
export type ResetPasswordState = {
  success?: boolean;
  error?: string;
  fieldErrors?: {
    password?: string;
    confirmPassword?: string;
  };
};
```

**resetPasswordAction implementation:**

1. Extract token, password, confirmPassword from formData
2. Validate password match - return fieldErrors.confirmPassword if mismatch
3. Validate password strength (min 8 chars, contains number) - return fieldErrors.password if weak
4. Validate token: `const validation = await validateResetToken(token)`
5. If `!validation.valid`, return: `{ error: "This reset link is invalid or has expired. Please request a new one." }`
6. Hash new password: `const hashedPassword = await hashPassword(password)`
7. Update user password: `await db.update(users).set({ password: hashedPassword }).where(eq(users.id, validation.userId))`
8. Mark token as used: `await consumeResetToken(validation.tokenId)`
9. Invalidate all other tokens: `await invalidateUserTokens(validation.userId)`
10. Return: `{ success: true }`
